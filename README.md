**Тестовое задание для Only** 

**О проекте:** 
- Адаптивная mobile-first верстка (мобильное устройство, десктоп) 
- Отзывчивые кнопки для лучшего UX 
- Слайдер с навигацией 
- Анимация выбора исторического периода 

**Стек технологий:** 
- React 
- Typescript 
- SCSS 
- Swiper 
- gsap 
 
**Как запустить проект:**  
- npm install 
- npm run start
